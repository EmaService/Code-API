# CodeApi — Todas tus IAs en un solo lugar

Plataforma unificada de acceso a modelos de IA con gestión de usuarios y API keys.

## Stack
- HTML / CSS / JS (single file)
- Supabase (auth + base de datos)
- Netlify (hosting)

## Deploy
Este proyecto se despliega automáticamente en Netlify desde la rama `main`.
