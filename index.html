Documentación de la API

Usa tu API key para autenticar todas las solicitudes. URL base: https://api.codeapi.io/v1

POST /chat

curl -X POST https://api.codeapi.io/v1/chat \ -H "Authorization: Bearer TU_API_KEY" \ -H "Content-Type: application/json" \ -d '{ "model": "claude-3-5-sonnet", "prompt": "Explica el aprendizaje automático", "max_tokens": 1000, "temperature": 0.7 }'
{ "id": "msg_capi_01XFDUDYJg", "model": "claude-3-5-sonnet", "content": "El aprendizaje automático es...", "usage": { "total_tokens": 150 }, "latency_ms": 89 }

JavaScript / Node.js

import { CodeApi } from '@codeapi/sdk'; const client = new CodeApi({ apiKey: process.env.CODEAPI_KEY }); const res = await client.chat.create({ model: 'claude-3-5-sonnet', prompt: 'Hola mundo' }); console.log(res.content);

Python

from codeapi import CodeApi client = CodeApi(api_key="TU_CODEAPI_KEY") response = client.chat.create( model="gpt-4o", prompt="Explica la recursión" ) print(response.content)
